Перелік усіх постійних виборчих дільниць в Україні, з координатами їх розташування та меж.
Дані скачано з сайту ДРВ - www.drv.gov.ua
1. oblast - назва регіону України;
2. okrug - номер виборчого округу;
3. popup - номер виборчої дільниці;
4. type - розмір виборчої дільниці;
5. address - адреса дільниці;
5. meji - опис меж дільниці;
6. prum - приміщення для голосування;
4. cord_X - координата широти;
5. cord_Y - координата довготи;
7. polygon - координати меж району ВД.
